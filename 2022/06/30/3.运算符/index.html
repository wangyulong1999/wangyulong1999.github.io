运算符

算数运算符

运算符 描述 实例
+ 加 10+20=30
- 减 30-10=20
* 乘 2*3=6
/ 除 6/3=2
// 取整除 5//2=2
% 取余除 9%2=1
** 幂 2 ** 3 = 8

在python中* 运算符还可以用于字符串列

比较关系运算符

运算符 描述
== 检查两个操作的值是否相等，如果是则条件成立，返回True
!= 检查两个操作的值是否不相等，如果是则条件成立，返回True
> 检查左边的数值是否大于右边的数值，如果条件成立，返回True
< 检查左边的数值是否小于右边的数值，如果条件成立，返回True
>= 检查左边测数值是否大于等于右边的数值，如果条件成立，返回True
<= 检查左边的数值是否小于等于右边的数值，如果条件成立，返回True

逻辑运算符

运算符 逻辑表达式 描述
and x and y 只有x和y返回都为True，返回才为True，x和y只要有一个False，都会返回False
or x or y 只要x和y有一个返回为True,返回为True，只有x和y都为False,才会返回False
not not x 如果x为True,则返回True,如果x为Flase，则会返回为True

赋值运算符

1.在python中，使用=可以给变量赋值

2.在算数运算符的时候，为了简写代码，python还提供了一系列的与算数运算符对应的赋值运算符

3.注意：赋值运算符中间没有空格

运算 描述 实例
= 赋值运算符 a = a + b 将a+b的结果赋值给c
+= 加等于赋值运算符 c += a 等效 c = c + a
-= 减等于赋值运算符 c -= a 等效 c = c - a
*= 乘等于赋值运算符 c * = a 等效 c = c * a
/= 除等于赋值运算符 c /= a 等效 c = c / a
//= 取整除赋值运算符 c //= a 等效 c = c // a
%= 区模等于赋值运算符 c %= a 等效 c = c % a
**= 幂赋值运算符 c * *= a 等效 c = c * * a

运算符的优先级

运算符 描述
** 幂（最高优先级）
* / % // 乘，除，取余除，取整除
+ - 加，减
<= < > > = 比较运算符
== != 等于运算符
= %= /= //= -= += *= **= 赋值运算符
not or and 逻辑运算符
